# Python DSA

Learning and implementing data structures and algorithms from scratch in Python.

## Purpose

This is a **hobby learning repo** where I explore computer science fundamentals through hands-on implementation. No deadlines, no pressure — just building things to understand how they work under the hood.

## What's Inside

### Data Structures
Core data structures implemented from scratch:
- Stacks (array-based & linked-list-based)
- Queues
- Linked Lists (singly & doubly)
- Trees (binary trees, BST)
- Graphs
- Hash Tables

### Algorithms
Classic algorithms coded and explained:
- Sorting (bubble, merge, quick, insertion)
- Searching (linear, binary)
- Recursion patterns
- Graph traversal (DFS, BFS)
- Dynamic programming basics

### Practice (optional)
Problems that use the concepts I've learned — mostly for reinforcement, not grinding.

## Approach

1. Learn a concept (e.g., "What's a stack?")
2. Implement it from scratch in Python
3. Understand how it works and when to use it
4. Add notes and examples
5. Move to the next topic when I feel like it

This isn't LeetCode prep — it's a reference repo for understanding fundamentals.
